1. Окно приложения меньше чем отображаемое содержимое приложения при открытии/создания текстового фильтра
Описание:Пользовательский интерфейс не полностью отображается при открытии или создании файла с фильтрами
Платформа: Windows8.1
Шаги:
1) Запустить приложение
2) Нажать "New" в меню File
Актульный результат: Окно приложения меньше чем отображаемое содержимое приложения
Ожидаемый результат: Окно приложения должно быть таким размером, чтобы содержимое приложения польностью отображалось
Серьезность:Trivial
Приложение: windowOfApp.png

2. Окно About зависает на время после нажатия на кнопку "Ok"
Описание:Окно About не отвечает несколько минут, затем закрывается
Платформа: Windows8.1
Шаги:
1) Запустить приложение
2) Нажать "About" в меню Help
3) Нажать "Ok"
Актульный результат: Окно About зависает на несколько минут
Ожидаемый результат: Окно About должно закрываться без зависания
Серьезность: Minor
Приложение: windowAbout.png

3. Новый шаблон файла не создается при нажатии на кнопку "New", когда уже есть отрытый файл
Описание: Новый пустой шаблон файла не создается при нажатии на кнопку "New", предыдущий открытый файл все еще открыт
Платформа: Windows8.1
Шаги:
1) Запустить приложение
2) Нажать "New" в меню File
3) Ввести данные в поле Element
4) Нажать кнопку "Add"
5) Нажать "New" в меню File

Актульный результат: Предыдущий открытый файл все еще открыт с введенными данными
Ожидаемый результат: Новый пустой шаблон файла открыт в новом окне
Серьезность:Major
